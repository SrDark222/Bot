## ⚔️ Visão Geral

> Bot Selfbot com IA persuasiva, que conversa como se fosse um usuário normal e convence os outros a entrarem no seu servidor sem parecer divulgação.

Esse projeto é o **infiltrado supremo** do Discord:  
Ele **responde como uma pessoa comum**, com escrita errada, minúscula, meio avoada, fazendo amizade e depois dando o bote — convida a pessoa pro server **sem levantar suspeita**.

---

## 🧠 Funcionalidades

- ✅ Respostas com **OpenRouter** ou **Gemini**
- ✅ Escrita humana (minúsculo, erros, sem vírgula)
- ✅ IA usa técnicas de persuasão e laço psicológico
- ✅ **Painel CLI** pra ativar/desativar IA, trocar modelo
- ✅ **Memória por usuário** (salva conversas antigas)
- ✅ Suporte ao Discord.js-Selfbot-v13
- ✅ Token **inserido manualmente** no terminal
- ✅ Pode mandar imagens com `catbox.moe`

---

## 🛠️ Instalação (Termux ou PC)

```bash
# 1. Clone o repositório
git clone https://github.com/seuuser/dk-discord-ai-bot
cd dk-discord-ai-bot

# 2. Instala dependências
pkg install nodejs -y   # no termux
npm install

# 3. Cria pasta de memória
mkdir memoria

# 4. Inicia o bot (ele vai pedir o token)
node index.js
